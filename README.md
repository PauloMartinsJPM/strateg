## Checklist de análise (antes de comprar qualquer ativo)
- P/VP < 1,0 → desconto  
- DY > 8% → bom | > 12% → cuidado com sustentabilidade  
- Vacância < 5% → saudável  
- Payout < 90% → sustentável  
- Dívida Líquida/EBITDA < 3x → segura
